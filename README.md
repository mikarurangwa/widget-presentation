# SegmentedButton Demo

A Flutter demo showcasing the **SegmentedButton** widget through a practical task filtering application.

## Widget Description
SegmentedButton is a Material 3 widget that allows single or multiple selection from a set of options, ideal for filters, toggles, and category selection.

## Run Instructions
1. Clone this repository
2. Run `flutter pub get`
3. Run `flutter run`

## Three Key Properties Demonstrated

1. **segments**: Defines the button options with labels, icons, and values
2. **multiSelectionEnabled**: Enables multiple simultaneous selections (true/false)
3. **showSelectedIcon**: Displays checkmark icon on selected segments (true/false)
